# Exec App

任意のシェルバイナリで、コマンドを実行できます。

## 概要

- ログを保存
- `mtk-su` など、任意のシェルバイナリを仕様
- ループを阻止

> [!WARNING]
> PoC を使っても、コンテキストの都合で全てのコマンドが実行できるとは限りません。

再起動や AppOps を実行したい場合、マニフェストにシステム権限を宣言し、システムアプリ化する必要があります。
